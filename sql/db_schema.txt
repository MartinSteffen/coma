1.  Tables:

 Table Conference
~~~~~~~~~~~~~~~~~~
id                            Int               Primary Key, Autoincrement
name                          Varchar 127	Not Null
homepage                      Varchar 127
description                   Text
abstract_submission_deadline  Date
paper_submission_deadline     Date
review_deadline               Date
final_version_deadline        Date
notification                  Date
conference_start              Date
conference_end                Date
min_reviews_per_paper         Int

 Table Person
~~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
first_name	Varchar 127
last_name	Varchar 127	Not Null
title		Varchar 32
affiliation	Varchar 127
email		Varchar 127	Unique, Not Null
phone_number	Varchar 20
fax_number	Varchar 20
street		Varchar 127
postal_code	Varchar 20
city		Varchar 127
state		Varchar 127
country		Varchar 127
password	Varchar 32      Not Null (encrypted)


 Table Role
~~~~~~~~~~~~~
conference_id	Int		-> Conference, id
person_id       Int             -> Person, id
role_type	Int		(enumeration Roles)
state		Int		(enumeration Status)

 Table Paper
~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
conference_id	Int		-> Conference, id
author_id	Int		-> Person, id
title		Varchar 127	Not Null
abstract	Text
last_edited	DateAndTime	(automatically set)
version		Int		(automatically set)
filename	Varchar 127
state		Int		(enumeration PaperState)
mime_type	Varchar 127

 Table IsCoAuthorOf
~~~~~~~~~~~~~~~~~~~~
person_id	Int
paper_id	Int		-> Paper, id
name		Varchar 127	Not Null

 Table Topic
~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
conference_id	Int		-> Conference, id
name		Varchar 127	Not Null

 Table IsAboutTopic
~~~~~~~~~~~~~~~~~~~~
paper_id	Int          	-> Paper, id
topic_id	Int		-> Topic, id

 Table PrefersTopic
~~~~~~~~~~~~~~~~~~~~
person_id	Int		-> Person, id
topic_id	Int		-> Topic, id

 Table PrefersPaper
~~~~~~~~~~~~~~~~~~~~
person_id	Int		-> Person, id
paper_id	Int          	-> Paper, id

 Table DeniesPaper
~~~~~~~~~~~~~~~~~~~
person_id	Int		-> Person, id
paper_id	Int             -> Paper, id

 Table ExcludesPaper
~~~~~~~~~~~~~~~~~~~~~
person_id	Int		-> Person, id
paper_id	Int             -> Paper, id

 Table ReviewReport
~~~~~~~~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
paper_id      	Int  		-> Paper, id
reviewer_id     Int	     	-> Person, id
summary         Text
remarks         Text
confidential    Text

 Table Rating
~~~~~~~~~~~~~~
review_id	Int		-> ReviewReport, id
criterion_id	Int		-> Criterion, id
grade		Int		Not Null
comment		Text

 Table Criterion
~~~~~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
conference_id	Int		-> Conference, id
name		Varchar 127     Not Null
description	Text
max_value	Int
quality_rating	Int		(in percent)

 Table Forum
~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
conference_id	Int		-> Conference, id
title		Varchar 127	Not Null
forum_type	Int		(enumeration ForumType)
paper_id	Int

 Table Message
~~~~~~~~~~~~~~~
id              Int             Primary Key, Autoincrement
forum_id	Int
reply_to	Int
sender_id	Int		-> Person, id
send_time	TimeAndDate	(automatically set)
subject		Varchar 127
text		Text


2. Enumerations (Vorschlag):

 Enumeration Roles
~~~~~~~~~~~~~~~~~~~~
01  admin
02  chair
03  reviewer
04  author
05  participant
...

 Enumeration Status
~~~~~~~~~~~~~~~~~~~~
01  active
02  not active
03  invited
...

 Enumeration PaperState
~~~~~~~~~~~~~~~~~~~~~~~~
01  being reviewed
02  conflicting
03  accepted
04  rejected
...

 Enumeration ForumType
~~~~~~~~~~~~~~~~~~~~~~~
01  open forum (for committee, authors and participants)
02  committee forum (only for chairs and reviewers)
03  paper discussion forum (only for committee members and authors
                            that are allowed to see the paper's review)
