\subsection*{Repository}

The repository resides on \texttt{snert}-machine. Write-access is
password-protected, for browsing the current status, you can click here:
\begin{center}
  \href{http://snert.informatik.uni-kiel.de:8080/svn/coma/}{\textbf{current repository status}}
\end{center}

The write access is granular in the following sense: each group can write
only within its own domain of responsibilities.
